# 上海地铁公共文化阵地

> 汇集上海地铁各站公共文化阵地展示效果的移动端平台

## 🌟 功能概览

- **首页轮播**：展示精选文化阵地
- **线路站点**：按线路筛选，快速查看各线站点
- **公益灯箱**：地铁空间公益宣传阵地
- **VR全景**：沉浸式站点艺术体验
- **演艺空间**：地铁站内的演出与艺术表演

## 📱 预览

访问 **https://shtz-culture.pages.dev** 查看在线演示

## 🛠 技术栈

- 纯 HTML + CSS + JavaScript
- 无框架依赖，轻量快速
- 响应式设计，适配移动端

## 🎨 设计参考

- UI 风格参考上海交响乐团（SSO）官方移动端
- 品牌色：交响红 `#c41e3a`
- 圆角卡片 + 模块化布局
